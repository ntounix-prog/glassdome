## Overview

**Target Users:**
- Security instructors (create and deploy scenarios)
- System administrators (monitor infrastructure)
- Students/trainees (access assigned labs)
- Auditors (compliance reporting)

### **API Communication**

```typescript
// API Client
import axios from 'axios';

const api = axios.create({
  baseURL: '/api/v1',
  timeout: 30000,
  headers: {
    'Content-Type': 'application/json',
  },
});

// WebSocket for real-time updates
const ws = new WebSocket('wss://glassdome/ws');

ws.onmessage = (event) => {
  const data = JSON.parse(event.data);
  
  switch(data.type) {
    case 'vm_status_update':
      updateVMStatus(data.payload);
      break;
    case 'deployment_progress':
      updateProgress(data.payload);
      break;
    case 'infrastructure_alert':
      showAlert(data.payload);
      break;
  }
};
```
